Summary
-------
This module provides an integrated framework for the design, simulation, and evaluation of hydrothermal liquefaction (HTL)-based waste valorization systems. It combines process simulation, machine learning, techno-economic analysis (TEA), life cycle assessment (LCA), uncertainty analysis, and multi-scale modeling modeling to evaluate the production of bio-based asphalt binders, sustainable aviation fuel (SAF) additives, and other valuable coproducts from diverse wet organic wastes. The module was developed as part of a project funded by `USDA <https://cris.nifa.usda.gov/cgi-bin/starfinder/0?path=fastlink1.txt&id=anon&pass=&search=R=98256&format=WEBLINK>`_.
